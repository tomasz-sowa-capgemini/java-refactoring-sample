# Gilded Rose Refactoring Kata (Java + C# subset)

This repository is a trimmed-down copy of the Gilded Rose Refactoring Kata, kept intentionally small to focus on Java and C# refactoring workflows.

## Upstream / Original Repository

- Original repo link: https://github.com/emilybache/GildedRose-Refactoring-Kata/tree/main
- Owner of the original repo: Emily Bache (GitHub: https://github.com/emilybache)

## What Changed in This Copy

- Removed other than Java and C# cases and other files: all other language implementations and supporting materials from the upstream kata were removed, keeping only the Java and C# variants present in this workspace.

## Purpose

The purpose of this repository is to test Java and C# AI-agentic refactoring (e.g., running an AI agent to refactor legacy code while continuously validating behavior with the existing tests).

## What’s Included

- C# implementations:
	- `csharp.NUnit/`
	- `csharp.xUnit/`
- Java implementations:
	- `Java/`
	- `Java-Approvals/`
	- `Java-Cucumber/`
	- `Java-Spock/`
